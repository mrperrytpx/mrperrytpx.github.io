GarbGarb

Summoner Lookup

A full-stack web application that allows you to search League of Legend players and provides you with access to their match history, champion statistics in ranked queues, information on whether they are currently engaged in a game, as well as details on any challenges they may have completed.

It's built using Create React App starter for the front-end with 'styled-components' as the styling framework and a Node Express.js server for the back-end. It utilizes JWTs for user authorization and authentication.

Serenity Hearth

An audio streaming website that allows you to stream an audio of a purring cat next to a crackling fireplace. You can also see the current amount of people streaming the audio. Inspired by Rainymood.

It's built using the Next.js framework for the front-end, TailwindCSS as the styling framework and a Node Express.js server for the back-end. It utilizes Socket.io to update the current number of listeners in real time via websockets.
